## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
   ```bash
   https://github.com/printlnkon/AGILA-REACT.git
   cd AGILA-REACT
   ```

2. Install dependencies
   ```bash
   cd frontend
   npm install
   ```

3. Start the development server
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`
